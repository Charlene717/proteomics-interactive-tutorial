空間蛋白質體學 (Spatial Proteomics)

把蛋白測量「定錨在組織座標上」——MALDI-IMS、LCM-microPOTS、Deep Visual Proteomics 的整合。

Anchor protein measurements to tissue coordinates — integrating MALDI-IMS, LCM-microPOTS, and Deep Visual Proteomics.

概念

為什麼需要空間？

傳統 bulk proteomics 將組織均質化後分析，遺失了腫瘤微環境、神經元層次、發育階段等空間結構資訊。空間蛋白質體學保留 X-Y（甚至 Z）座標，回答「哪些蛋白只在腫瘤侵犯前緣、不在中心？」這類問題。

當前三大主流策略：(1) MALDI-IMS（無偏好成像，但 ID 數有限）；(2) LCM-microPOTS（先用 LCM 切下感興趣區，再做深度 MS）；(3) Deep Visual Proteomics（AI 自動辨識細胞型態，再針對性 sampling）。

Bulk proteomics homogenizes tissue and loses spatial structure — tumor microenvironment, neuronal layers, developmental stages. Spatial proteomics preserves X-Y (and sometimes Z) coordinates to answer "which proteins are restricted to the invasive tumor front but not the core?"

Three current mainstream strategies: (1) MALDI-IMS (unbiased imaging, limited IDs); (2) LCM-microPOTS (laser-capture microdissect ROIs, then deep MS); (3) Deep Visual Proteomics (AI identifies cell morphologies, then targeted sampling).

三大方法

一、三大空間蛋白質體學方法

🌈 MALDI-IMS

樣品切片噴 matrix → 雷射逐格 ablate → 每格產生 MS1 譜 → 重建分子分布影像。
• 解析度：5–100 μm (空間)；目前最快可達 single-cell (~5 μm)
• 偵測：胜肽、完整蛋白或裂解片段
• 限制：受限離子化效率，ID 數通常 < 1000

Spray matrix on tissue section → laser ablates pixel-by-pixel → MS1 spectrum per pixel → reconstruct molecular distribution.
• Resolution: 5–100 μm spatial; best now ~single-cell (~5 μm)
• Detects: peptides, intact proteins, or lipid fragments
• Limit: Lower ionization efficiency; usually < 1000 IDs

✂️ LCM-microPOTS / nanoPOTS

用 LCM (Laser Capture Microdissection) 切下 50–500 個細胞區域 → 在 nanowell 中做 nanoPOTS 樣品製備 → LC-MS/MS 深度 ID。
• 解析度：10–200 μm（依 LCM 切割）
• 偵測：數百到 5000+ 蛋白/區域
• 適用：已知區域邊界（如腫瘤 vs 健康）

LCM (Laser Capture Microdissection) cuts 50–500-cell ROIs → nanoPOTS prep in nanowells → LC-MS/MS deep ID.
• Resolution: 10–200 μm (depends on LCM)
• Detects: hundreds to 5000+ proteins per ROI
• Use: Known region boundaries (tumor vs healthy)

🤖 Deep Visual Proteomics

2022 由 Mann lab 提出。流程：染色組織 → AI/Deep Learning 自動分類細胞型態 → 選擇 phenotype → 自動化 LCM → MS 深度 proteomics。
• 核心優勢：無需手動勾勒邊界，可分析罕見 cell type
• 應用：肝癌、肌營養不良、發育生物學

Proposed by the Mann lab in 2022. Workflow: stain tissue → AI/deep learning classifies cell phenotypes → choose phenotype → automated LCM → deep proteomics.
• Key advantage: No manual boundary drawing; rare cell types analyzable
• Applications: Hepatocellular carcinoma, muscular dystrophies, developmental biology

🎯 MALDI-HiPLEX-IHC (2024)

2024 新方法：將抗體標記上光裂解質量標籤 → 同時做 IHC 染色 + MS 成像。可一次顯像 30+ 蛋白的空間分布，補足 MALDI-IMS 偏向 abundant proteins 的弱點。

2024 method: antibodies labeled with photocleavable mass tags → simultaneous IHC + MS imaging. Visualizes 30+ proteins' spatial distribution at once, complementing MALDI-IMS's abundance bias.

案例

二、實際案例

🫀 心肌組織異質性

2024 MCP 研究：MALDI-IMS + LC-MS/MS 結合分析心肌纖維化區域，發現 STAT3 與 collagen IV 在纖維化前緣高度共定位。

A 2024 MCP study used MALDI-IMS + LC-MS/MS on cardiac fibrosis regions, showing STAT3 and collagen IV co-localize at the fibrotic front.

🧠 腦部細胞型態

使用 DVP 自動分類 cortex 神經元、astrocyte、oligodendrocyte，發現 oligodendrocyte 的特異蛋白 30% 在 bulk 數據中完全被淹沒。

DVP automatically classified cortical neurons, astrocytes, and oligodendrocytes; 30% of oligodendrocyte-specific proteins were entirely masked in bulk data.

🧬 腫瘤微環境

LCM-microPOTS 對乳癌 tumor / stroma / immune 三區分別深度 proteomics，建構區域特異性 biomarker 圖譜。

LCM-microPOTS profiled tumor / stroma / immune compartments in breast cancer separately, building region-specific biomarker atlases.

🐠 發育時間軸

斑馬魚胚胎不同階段的空間蛋白質體學，揭露 organogenesis 的關鍵分子事件。

Zebrafish embryo spatial proteomics across stages revealed key molecular events of organogenesis.

# imzML 是 MALDI-IMS 的標準格式
img <- readMSIData("section.imzML")

# Pre-processing
img <- normalize(img, method = "tic") %>%
       smoothSignal(method = "gaussian", window = 5) %>%
       reduceBaseline(method = "locmin") %>%
       peakAlign(tolerance = 200, units = "ppm")

# Spatial segmentation (unsupervised)
seg <- spatialShrunkenCentroids(img, r = 2, k = 5, s = 0:5)
image(seg, model = list(s = 3))  # spatial maps

# Per-cluster markers
top_features <- topFeatures(seg, model = list(s = 3), n = 20)
