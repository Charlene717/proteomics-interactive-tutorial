# Proteomics Interactive Tutorial

**🌐 Live demo / 線上瀏覽**: <https://charlene717.github.io/proteomics-interactive-tutorial/>


完整的雙語（繁中／English）蛋白質體學互動式教程，涵蓋現代蛋白質體學的核心管線與進階主題。

## 內容架構

**核心管線（9 步驟）**：
1. Sample Preparation & Digestion
2. LC-MS/MS Acquisition (DDA / DIA)
3. Database Search & Identification
4. Quantification (LFQ / TMT / SILAC)
5. Quality Control
6. Normalization
7. Missing Value Imputation
8. Differential Expression
9. Functional Enrichment

**進階主題（6 篇）**：
- PTM (Phosphoproteomics)
- Targeted Proteomics (PRM/MRM)
- Top-down Proteomics
- Spatial Proteomics
- Single-cell Proteomics
- Affinity Proteomics (Olink / SomaScan)

## 互動元素
- Chart.js 即時模擬
- R / Python / CLI / GUI 多工具程式碼分頁
- Self-check Quiz 自我檢測
- 雙語即時切換（localStorage 記憶）
