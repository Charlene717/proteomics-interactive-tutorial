功能富集分析 (Functional Enrichment)

從蛋白清單到生物學故事——ORA、GSEA、STRING PPI 三種互補視角。

From a protein list to a biological story — ORA, GSEA, and STRING PPI as three complementary lenses.

兩大思路

一、ORA vs GSEA 的本質差異

🎯 ORA — Over-Representation

「我先把蛋白分成『顯著』與『不顯著』兩堆，再問每個 pathway 中『顯著』蛋白的比例是否超出隨機期望？」
用 Fisher's exact test 或 hypergeometric test。
• 適用：已知 cutoff、結果清單明確的場合
• 限制：會「丟掉」次顯著但具趨勢的蛋白；對 cutoff 敏感

"Split proteins into significant vs not, then ask whether each pathway is over-represented in the significant pile."
Tests with Fisher's exact or hypergeometric.
• Use: When you have a clear cutoff and confident hit list
• Limit: Throws away sub-significant trends; sensitive to cutoff

📈 GSEA

「全部蛋白按某個指標（如 log2FC * -log10(p)）排序，問每個 pathway 的成員是否『傾向集中在排序的某一端』。」
用 Kolmogorov-Smirnov 統計量 + permutation。
• 適用：連續訊號、未必有明顯 cutoff、要捕捉協調性變化
• 限制：需要 ranked list 而非 binary hit list；解釋複雜

"Rank all proteins by a metric (e.g., log2FC × -log10(p)) and ask whether each pathway's members cluster at one end."
Uses KS statistic + permutation.
• Use: Continuous signal, no clear cutoff, capturing coordinated changes
• Limit: Requires a ranked list, not a binary hit list; trickier to interpret

💡
實務建議：兩者並行——ORA 給出「強訊號 pathway」，GSEA 給出「整體趨勢 pathway」。最重要的是仔細選擇 background：必須是「實驗中所有被測到的蛋白」，而非整個 UniProt 資料庫，否則會大幅膨脹假陽性。 Practice: Run both — ORA gives "strong-signal pathways", GSEA gives "overall trend pathways". The most important detail is the background: it must be all proteins detected in your experiment, not the full UniProt — otherwise false positives explode.

四、選擇 enrichment 工具的決策

🌳 enrichment 決策樹

Q1:
已有明確 hit list（cutoff）？ → 是 → ORA (clusterProfiler / g:Profiler / Enrichr)。
Q2:
想看「整體趨勢」、不希望 cutoff 卡死？ → 是 → GSEA / fgsea，用 signed -log10(p) 或 log2FC 作 ranks。
Q3:
想看「蛋白如何互相連結」、找 hub 蛋白？ → 是 → STRING + Cytoscape。
Q4:
要可發表的 figure？ → 是 → EnrichmentMap (Cytoscape app) 是 Nature Protocols 推薦標準。
Q1:
Clear hit list with a cutoff? → Yes → ORA (clusterProfiler / g:Profiler / Enrichr).
Q2:
Want the overall trend without a hard cutoff? → Yes → GSEA / fgsea with signed -log10(p) or log2FC as ranks.
Q3:
Want to see protein interconnections and hubs? → Yes → STRING + Cytoscape.
Q4:
Need a publication-grade figure? → Yes → EnrichmentMap (Cytoscape app) — the Nature Protocols standard.
